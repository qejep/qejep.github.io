------------------------------
главная ⇢ читательский маршрут
------------------------------

Спики книг, представленные здесь, помогают наметить читательские маршруты. Однако они не являются ни исчерпывающими, ни обязательными. Это просто тематические подборки текстов, которые я прочёл или планирую прочесть.
  1. Список книг, для тех, кто хочет подружиться с головой.
  2. Список книг, в которых художественно осмысляется феномен рекламы и PR.
  3. Список книг, для тех, кто хочет научиться писать прозу.
  4. Спискок книг, для тех, кто хочет научиться писать статьи, эссе, в общем, нонфикшен.
  5. Список книг, для тех, кто хочет выучить много иностранных языков.
  6. Список книг к курсу по философской герменевтике (предупреждаю, что список длинный!).
  7. Список книг по библейской герменевтике.
